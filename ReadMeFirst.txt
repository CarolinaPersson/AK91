Task 4.1 Version control on GitHub for AK91 Code.

This is folder Task4/AK91.

This forlder contains the master program (do file), a project map (Excel), a gitignore file, 
a git folder, a backup.bat file and three subfolders.

1. The Raw folder contains the raw data New7080.dta, downloaded from http://economics.mit.edu/faculty/angrist/data1/data

2. The Build folder contains two subfolders: the Build/Input folder and the Build/Code folder.

The Build/Code folder includes the code for the cleaning and creation of new variables. The cleaned dataset is saved in the
Analysis/input folder

3. The Analysis folder contains three subfolders: the Analysis/Input, the Analysis/Code and the Analysis/Output folders.
